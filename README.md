# agentic-coding-stories-from-the-field

Area to collect ideas and experiences around agentic coding, with the goal of crafting a final blog post.

## Agents

This repository includes two GitHub Copilot custom agents to assist with content creation:

- **[Technical Writer](/.github/agents/technical-writer.md)** – Helps turn raw ideas, notes, and experiences into polished, structured blog content.
- **[AI SME](/.github/agents/ai-sme.md)** – Provides expert knowledge on agentic coding, LLMs, and AI-assisted development to ensure technical accuracy and depth.

Use `@technical-writer` to draft and refine content, and `@ai-sme` to validate technical accuracy and explore AI concepts in depth.
